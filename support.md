# Nurr Support

Last updated: April 10, 2026

This page provides support information for Nurr.

## About the App

Nurr is a Quran and dua app with:

- Quran reading views
- Mushaf reading pages
- bookmarks and last-read position
- dua content
- tasbih-related local app state
- prayer tracking
- location-based prayer times
- multilingual app support

## Support Topics

You can request help with:

- app installation or launch issues
- Quran, Mushaf, or page-loading problems
- prayer times not loading
- location permission issues
- bookmark or saved-page issues
- dua or content display issues
- translation or language issues
- theme or display issues
- general bug reports and feature requests

## Basic Troubleshooting

If something is not working, try the following:

1. Close and reopen the app.
2. Check that your internet connection is working.
3. Make sure location permission is enabled if you want to use prayer times.
4. On web, allow browser location access if prompted.
5. Try again later if a third-party content or prayer-time service is temporarily unavailable.
6. Update to the latest app version if one is available.
7. If local saved data appears broken, reinstalling or clearing app storage may reset saved settings and progress, depending on your platform.

## Prayer Times Support

The prayer times feature depends on location access and external services.

- If you deny location permission, prayer times may not load correctly.
- On web, if direct location access fails or is unavailable, the app may use an IP-based location fallback.
- Results may vary slightly depending on the external prayer-time provider, calculation method, and location accuracy.

## Content Availability

Some app content depends on external services used to load Quran, Hadith, or prayer-related data.

- If a third-party service is unavailable, some content may fail to load temporarily.
- Network restrictions, browser settings, or provider-side issues may affect content loading.
